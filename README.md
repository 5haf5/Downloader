# YouTube Downloader – návod k použití

Tato aplikace umožňuje stahovat videa i hudbu z YouTube pomocí jádra yt-dlp.

---

## ▶ Jak spustit program
Spusťte soubor:
- Downloader.exe

Doporučeno: vytvořte si zástupce na plochu, který bude odkazovat na Downloader.exe ve složce *Release*.

---

## 🎵 Funkce aplikace
- Stahování videí v různých kvalitách (360p–4320p)
- Stahování pouze hudby (MP3)
- Výběr cílové složky
- Logování průběhu stahování
- Automatická aktualizace jádra (yt-dlp)

---

## 🔄 Aktualizace yt-dlp
Pravidelně spouštějte aktualizaci jádra.

V aplikaci klikněte na:
**Aktualizovat jádro (yt-dlp)**

Aplikace automaticky provede příkaz:
yt-dlp -U

---

## 📁 Struktura složky
Aplikace vyžaduje tyto soubory:
- Downloader.exe
- yt-dlp.exe
- ffmpeg.exe
- ffprobe.exe
- další .dll soubory, které jsou součástí složky Release
